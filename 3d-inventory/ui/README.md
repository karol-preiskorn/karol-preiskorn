## Overview

The 3D Inventory Angular UI is a comprehensive frontend application designed to create spatial and database representations of warehouses and datacenters with interactive 3D visualization.

## Key Features

- **3D Visualization**: Interactive Three.js-based floor plans
- **Device Management**: Create, edit, and position devices spatially
- **Role-Based Access**: Admin, User, and Viewer permission levels
- **Activity Monitoring**: Comprehensive audit trails with change tracking
- **reCAPTCHA v3**: Invisible bot protection on login

## Technology Stack

- **Framework**: Angular 20.3+ (standalone components)
- **UI**: Bootstrap 5.3+
- **3D Graphics**: Three.js 163+
- **Testing**: Jest with comprehensive coverage
- **Deployment**: Docker + Google Cloud Run

## Getting Started

```bash
# Clone the repository
git clone https://github.com/karol-preiskorn/3d-inventory-angular-ui.git
cd 3d-inventory-angular-ui

# Install dependencies
npm install

# Start development server
npm run start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.
