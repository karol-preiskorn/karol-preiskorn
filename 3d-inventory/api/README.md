## Overview

The 3D Inventory Mongo API is a comprehensive backend solution for managing inventory systems with spatial visualization capabilities.

## Key Features

- **JWT Authentication** with Role-Based Access Control (RBAC)
- **RESTful API** with OpenAPI/Swagger documentation
- **MongoDB Atlas** integration with connection pooling
- **Docker** containerization for deployment
- **Comprehensive logging** with audit trails

## Technology Stack

- **Runtime**: Node.js 22+ with Express.js
- **Language**: TypeScript (strict mode)
- **Database**: MongoDB Atlas
- **Testing**: Jest with >80% coverage
- **Deployment**: Google Cloud Run

## Getting Started

```bash
# Clone the repository
git clone https://github.com/karol-preiskorn/3d-inventory-mongo-api.git
cd 3d-inventory-mongo-api

# Install dependencies
npm install

# Start development server
npm run dev
```
